# GoogleClone
Clonación de la interfaz de Google...Creado con HTML5 Y CSS3 en el bootcamp de TecnolochicasPro.

## 📚 Intro
HTML5 o HyperText Markup Language, es el componente básico de la web y junto con CSS3, se crean páginas web que usamos todos los días en el navegador. Definimos el significado, la estructura y el estilo del contenido web a través de estas tecnologías. 
En este proyecto, se utilizó HTML5 y CSS3 para construir una clonación de la interfaz de Google.

## 👩🏽‍💻 ¿Qué construí?
El proyecto está enfocado en la construcción de la interfaz de Google, tomando especial atención a su UI/UX para replicar su estructura y estilo. La interfaz se encuentra dividida en las siguientes secciones: 

* Header: Sección donde se encuentra la navegación principal de Google, como la foto de perfil del usuarix, íconos y menú. 

* Main: Sección del contenedor para los elementos centrales y más importantes de la interfaz: logo, barra de búsqueda, íconos y botones. 

* Footer: Sección que contiene información adicional para el usuarix, así como hipervínculos al final de la página. 

## 🚀 Objetivo del Proyecto.
Aprender a utilizar las etiquetas de HTML5 y las propiedades de CSS3; así como identificar componentes base de un sitio web, para analizarlas y replicarlas.
